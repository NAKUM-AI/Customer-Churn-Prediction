# Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to leave a bank using customer information such as age, balance, credit score, geography, and account activity.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## 📌 Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Accuracy Comparison
- Confusion Matrix
- ROC Curve

## 📂 Dataset
**Churn_Modelling.csv**
